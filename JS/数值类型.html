1. 几种数值类型： number、 string 、 undefined、 null、 Boolean、symbol 2. 判断类型： (1) typeof : null是object， 判断数组和对象是也是object,判断函数返回function (2) instanceof ：XX是XX的一个实例。a=new A(), a instanceof A 返回true。 原理是 Object.getPrototypeOf(a) = A.prototype。 不能识别出基本数据类型, 通常用来检测对象类型，以及判断继承关系 (3) Object.prototype.toString.call(): 对于所有的数据类型都能判断，包括undefined和null (4) Array.isArray() : 判断数组 (5) constructor: 不安全，因为constructor的指向可能被改变
