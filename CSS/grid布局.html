容器属性： 1. display: grid 2. grid-template-rows: 每行多宽 100px 50px 30px grid-template-columns: 每列多宽 30px 40px 50px (1)repeat(times, width/height): 接收两个参数 (2)auto-fill: 自动填充。当容器大小不固定时，该方法可以自动填充容器。repeat(auto-fill, 100px) (3)指定网格线名称：grid-template-rows:[r1] 100px [r2] 100px [r3] 100px [r4] grid-template-columns:[c1] 100px [c2] 100px [c3] 100px [c4] 3.row-gap/column-gap/gap： 设置行单元格或者列单元格之间的间距 4.分割区域：grid-template-areas 如果有些区域不需要利用，则用.来表示 5.grid-auto-flow: column/row/ row dense/ column dense 按行/列进行自动填充、补全 6.justify-items :start/ end / center/ stretch 设置每个单元格的水平/垂直位置 align-items 7.justify-content/ align-content: start/end/center/ stretch/ space-around/space-between 设置单元格整体的水平、垂直位置 项目属性： 1.指定项目的具体位置。具体方法就是指定项目的4个边框，分别所在的4条网格线 grid-column-start/ grid-column-end/ grid-row-start/ grid-row-end。 简写为grid-column grid-row 2.指定项目放在哪个区域 grid-area:1 3. justify-self/align-self: 只作用于单个单元格。设置水平 垂直位置
