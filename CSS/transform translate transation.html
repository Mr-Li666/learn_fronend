transform：translate/rotate(20deg)/scale()/skew 对元素进行移动、旋转、拉伸、缩放、倾斜转换 transition: 元素从一种状态过渡为另一种状态 transition: property/duration/transition-function/delay property: 要产生过度效果的属性名 width height color duration: 设置多少秒完成过渡效果。时长为0时，不会产生过渡效果。 transition-function:以何种方式进行过度。 默认ease：慢速开始，然后变快，然后慢速结束的过渡效果。 ease-in:规定以慢速开始的过度效果 ease-out：规定以慢速结束的过度效果 liner：以相同的速度从开始至结束完成过度 delay: 在等待多少秒之后 开始执行过度
